3 Findings

This section might be called “Results” instead of “Applications,” depending on what it is that you are working on. But you’ll probably say something like “The initial model estimation results are given in Table 3.1.” That table is created with the modelsummary() package and function.

model1 <- mlogit(choice ~ type + price | -1, data = car_mlogit)
model2 <- update(model1, .~. + range)

models <- list("Model 1" = model1, "Model 2" = model2)
modelsummary(models, estimate = "{estimate} ({statistic})",
             title = "Model Estimation Results")
